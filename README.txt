اسم المشروع:
Chef Finder – ابحث عن الطباخ المثالي لمناسبتك

وصف المشروع:
موقع ويب يتيح للمستخدم البحث عن طباخين محترفين حسب نوع المطبخ
والاطلاع على بياناتهم التفصيلية مع إمكانية حجز موعد.

التقنيات المستخدمة:
- HTML5
- CSS3
- JavaScript
- JSON (كمحاكاة لقاعدة بيانات)
- Back-End بسيط (Concept)

هيكل المشروع:
- index.html : الصفحة الرئيسية
- pages/ : صفحات الأقسام وتفاصيل الطباخين
- css/ : ملفات التنسيق
- js/ : ملفات البرمجة
- images/ : الصور
- backend/ : محاكاة الخادم وقاعدة البيانات

فكرة الحجز:
يتم اختيار تاريخ ووقت، ويقوم JavaScript بالتحقق من الإدخال
وإظهار رسالة نجاح الحجز كمحاكاة لنظام حجز فعلي.

ملاحظات:
تم الالتزام بتنظيم الملفات وعدم وضع الكتاب داخل ملف ZIP
طبقًا لتعليمات المشروع.
